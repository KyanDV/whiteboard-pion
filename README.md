# Collaborative Whiteboard (Golang WebRTC)
Project ini adalah aplikasi papan tulis digital *real-time*. Pengguna bisa menggambar, menghapus, dan menulis teks bersama-sama dalam satu kanvas.

Untuk menjalankan program ini:
1. Download project ini sebagai zip
2. Ekstrak zip di folder
3. Kemudian buka folder tersebut di Visual Studio dan jalankan perintah 'go mod tidy' di terminal VS.
4. Dan jalankan perintah 'go run main.go' di terminal
5. Tekan ctrl + click pada http://localhost:8080 di terminal, lakukan 2 kali untuk membandingkan hasil gambar dari 2 tab berbeda.
